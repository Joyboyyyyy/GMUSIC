RUNNING THE PROJECT:
====================

1. INSTALL ALL DEPENDENCIES:
   npm run install:all

2. START DEVELOPMENT (both frontend & backend):
   npm run dev

3. START ONLY FRONTEND:
   npm run dev:frontend

4. START ONLY BACKEND:
   npm run dev:backend

5. BUILD FRONTEND:
   npm run build:frontend

6. CLEAN ALL NODE_MODULES:
   npm run clean


ENVIRONMENT VARIABLES:
======================

Root .env:
  - Shared configuration if needed
  - Can be used by both frontend and backend

frontend/.env:
  - Frontend-specific variables
  - API endpoints, feature flags, etc.

backend/.env:
  - Backend-specific variables
  - Database URLs, API keys, secrets, etc.
