RUNNING GRETEX MUSIC ROOM - SEPARATE PROJECTS
==============================================

PROJECT STRUCTURE:
==================

frontend/          - React Native mobile app (completely independent)
backend/           - Node.js Express API (completely independent)
docs/              - Documentation
.kiro/             - Kiro specs
.vscode/           - VS Code settings


RUNNING FRONTEND:
=================

1. Open Terminal 1
2. Navigate to frontend:
   cd frontend

3. Install dependencies (first time only):
   npm install

4. Start development server:
   npm start

5. Choose platform:
   - Press 'a' for Android
   - Press 'i' for iOS
   - Press 'w' for Web


RUNNING BACKEND:
================

1. Open Terminal 2
2. Navigate to backend:
   cd backend

3. Install dependencies (first time only):
   npm install

4. Set up environment variables:
   - Copy .env.example to .env (if exists)
   - Update with your configuration

5. Start development server:
   npm run dev


RUNNING BOTH SIMULTANEOUSLY:
============================

Option A: Two Terminal Windows
  Terminal 1: cd frontend && npm start
  Terminal 2: cd backend && npm run dev

Option B: Using VS Code
  - Open gretex-music-room.code-workspace
  - Open integrated terminal
  - Split terminal (Ctrl+Shift+5)
  - Run both commands side by side


ENVIRONMENT SETUP:
==================

Frontend (.env):
  REACT_APP_API_URL=http://localhost:5000
  REACT_APP_FEATURE_FLAG=true

Backend (.env):
  PORT=5000
  DATABASE_URL=your_database_url
  JWT_SECRET=your_secret_key


TROUBLESHOOTING:
================

Frontend won't start:
  - Delete node_modules: rm -r frontend/node_modules
  - Clear cache: npm cache clean --force
  - Reinstall: npm install

Backend won't start:
  - Check .env file exists
  - Verify database connection
  - Check port 5000 is not in use

Port conflicts:
  - Frontend: Usually runs on port 19000 (Expo)
  - Backend: Change PORT in .env if 5000 is taken


QUICK COMMANDS:
===============

Frontend:
  npm install          - Install dependencies
  npm start            - Start dev server
  npm run android      - Build for Android
  npm run ios          - Build for iOS
  npm run web          - Run on web

Backend:
  npm install          - Install dependencies
  npm run dev          - Start dev server
  npm run build        - Build for production
  npm run seed         - Seed database
  npm test             - Run tests
